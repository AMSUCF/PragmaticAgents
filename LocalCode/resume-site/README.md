# Resume Website

A beautiful, artistic pastel-colored resume website for Some Student.

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop devices
- **Artistic Pastel Color Scheme**: Soft, elegant colors with a creative touch
- **Interactive Skills Visualization**: Animated proficiency chart
- **Complete Resume Content**: Education, skills, experience, projects, and activities
- **Downloadable PDF**: Option to download resume as PDF

## Structure

```
resume-site/
├── index.html          # Main resume page
├── about.html          # About page (placeholder)
├── contact.html        # Contact page (placeholder)
├── css/
│   └── style.css       # Main stylesheet with pastel color scheme
├── js/
│   └── skills-chart.js # Interactive skills visualization
├── pdf/
│   └── resume.pdf     # Downloadable resume PDF
├── images/             # Image assets (empty)
└── test.html           # Test page for verification
```

## Color Palette

- Primary: #8B5A8C (Soft Purple)
- Secondary: #6B8E98 (Soft Blue-Gray)
- Accent: #E8A5A5 (Soft Pink)
- Accent 2: #B8E6B8 (Soft Green)
- Background: #F8F9FA (Off White)

## How to Use

1. Open `index.html` in any web browser
2. Navigate through the resume sections
3. Download the PDF version using the download button

## Browser Support

- Chrome/Edge
- Firefox
- Safari
- Opera

## Responsive Breakpoints

- Mobile: < 480px
- Tablet: 480px - 768px
- Desktop: > 768px

## Accessibility

- Semantic HTML structure
- Proper contrast ratios
- Keyboard navigable
- Screen reader friendly
